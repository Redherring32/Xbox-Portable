This is a preliminary version of the Xbox Power Management System used in this portable Xbox design.
It does not yet have charging built in, but can be used in its current state as a compact power supply for revision 121 Xbox motherboards that have both the stock regulators removed, as well as standby disabled.

For info on identifying a 121 Xbox PCB you can read this guide here: https://www.herringhardware.com/XboxRevisions.html
For info on disabling standby and removing stock regulators on a 121 motherboard you can read this guide: https://bitbuilt.net/forums/index.php?threads/xbox-trimming-guide.6157/
